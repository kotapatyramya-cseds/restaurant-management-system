# Restaurant Management System

## Project Overview

Restaurant Management System is a web-based application developed using PHP and MySQL to manage restaurant operations efficiently. The system provides separate modules for Admin, Cashier, and Customer management.

## Features

- Order Management
- Billing System
- Customer Management
- Admin Dashboard
- Payment Tracking
- Order Reports
- User Authentication

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Modules

### Admin
- Manage orders
- Manage customers
- View reports

### Cashier
- Process orders
- Generate bills
- Track payments

### Customer
- Place orders
- View order history
- Manage profile
